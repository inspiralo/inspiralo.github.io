# inspiralo.github.io
Forráskódokat kutatni imádó alkat vagyok, amely különösen a kriptográfiai úton kezelt vagyonelemek során számomra önvédelem. Látnom kell mi decentralizált, s mi valami centralizált huncutság. Inkább tanulmányozok. A köz javára ugyan nem fejlesztek, de magamnak sok mindent megoldok, alapvetően ismeretterjesztéssel, tartalomfejlesztéssel foglalkozok.
